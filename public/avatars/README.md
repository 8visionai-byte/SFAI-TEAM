# Awatary agentów

Tu wrzucasz gotowe obrazki awatarów person. Aplikacja podłapie je sama,
bez zmian w kodzie.

## Jak nazwać pliki

- Nazwa pliku: `<slug>.png` (dokładnie taki slug jak niżej, małe litery).
- Format: PNG, kwadrat (najlepiej 512x512 px, po optymalizacji cel poniżej 150 KB).
- Brak pliku niczego nie psuje: aplikacja pokazuje wtedy inicjały na tle
  koloru agenta, tak jak do tej pory.

| Plik | Persona |
|---|---|
| `coo.png` | COO (orkiestrator) |
| `wiedza-produkt.png` | Specjalista od wiedzy i materiałów |
| `operacje.png` | Asystent od ogarniania (operacje) |
| `analityk.png` | Analityk rynku |
| `pamiec-zespolu.png` | Pamięć całego zespołu |
| `copywriter.png` | Copywriter marki |
| `handlowiec.png` | Handlowiec od korzyści |
| `opiekun-klienta.png` | Opiekun klienta |
| `drugi-glos.png` | Drugi głos przy decyzjach |
| `analityk-social.png` | Analityk Social Mediów (persona w przygotowaniu, plik możesz wrzucić już teraz) |

Prompty do wygenerowania obrazków: plik `AWATARY-HIGGSFIELD-PROMPTY.md`
w głównym katalogu projektu.
